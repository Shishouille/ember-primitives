# Iframe

Utilities for working with IFrames.

## Installation 

```bash 
pnpm add ember-primitives
```


### `inIframe`

```gjs live no-shadow
import { CommentQuery } from 'docs-app/docs-support';

<template>
  <CommentQuery @module="iframe" @name="inIframe" />
</template>
```

### `notInIframe`

```gjs live no-shadow
import { CommentQuery } from 'docs-app/docs-support';

<template>
  <CommentQuery @module="iframe" @name="notInIframe" />
</template>
```
